# TODO List com NGRX/Signals

Esse projeto é uma implementação de uma lista de tarefas utilizando o NGRX/Signals do Angular. Todas as informações são salvas em localstorage.

O NGRX/Signals é uma biblioteca que fornece uma forma fácil de gerenciar e lidar com eventos em uma aplicação Angular. Ela fornece uma forma de publicar e inscrever em eventos, permitindo que os componentes sejam notificados quando um evento é disparado.

Nesse projeto, as tarefas são salvas em localstorage e quando o usuário adiciona, remove ou altera uma tarefa, um evento é disparado e todos os componentes que se inscreveram nesse evento são notificados, atualizando a lista de tarefas.
